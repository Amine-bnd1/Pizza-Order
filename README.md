# Pizza-Order

A simple **C# Windows Forms** application for building and ordering a custom pizza.  
Users can choose the pizza **size**, **toppings**, **crust type**, and **where to eat**, while the app updates the **order summary** and **total price** automatically.

## Features

- Select pizza size:
  - Small
  - Medium
  - Large
- Choose multiple toppings:
  - Extra Cheese
  - Onion
  - Mushrooms
  - Olives
  - Tomatoes
  - Green Peppers
- Select crust type:
  - Thin Crust
  - Thick Crust
- Choose dining option:
  - Eat In
  - Take Out
- Real-time order summary updates
- Automatic total price calculation
- Order confirmation dialog
- Reset form to start a new order

- **C#**

## How It Works

The application allows the user to customize a pizza order through a graphical interface.

### 1. Pizza Size
The user selects one pizza size:
- Small
- Medium
- Large

Each size has its own base price.

### 2. Toppings
The user can add one or more toppings.  
Each selected topping increases the total price.

### 3. Crust Type
The user selects one crust type:
- Thin Crust
- Thick Crust

The crust selection may affect the final price.

### 4. Where to Eat
The user chooses whether the order is:
- Eat In
- Take Out

### 5. Order Summary
The application displays:
- Selected size
- Selected toppings
- Selected crust type
- Dining option
- Total price

### 6. Order Confirmation
When the user clicks **Order Pizza**, the application asks for confirmation.  
If confirmed:
- A success message is shown
- The form controls are disabled
- The order button becomes unavailable until the form is reset

### 7. Reset
The **Reset Form** button restores the default selections and allows placing a new order.

## Pricing Logic

The total price is calculated from:

- **Base size price**
- **Sum of selected toppings**
- **Crust price adjustment**

### Default pricing in the current implementation
- Small: **$20**
- Medium: **$30**
- Large: **$40**
- Each topping: **$5**
- Thin crust: **$0**
- Thick crust: **$10**

## Getting Started

### Prerequisites

Make sure you have:

- **Visual Studio 2022** or later recommended
- **.NET Framework 4.8 Developer Pack**

### Run the project

1. Clone the repository:
   ```bash
   git clone https://github.com/Amine-bnd1/Pizza-Order.git
   ```

2. Open the solution:
   - Open `Pizza.sln` in Visual Studio

3. Build the project:
   - Select **Build Solution**

4. Run the application:
   - Press **F5** or click **Start**

## Usage

1. Launch the app
2. Choose a pizza size
3. Select one or more toppings
4. Choose the crust type
5. Select whether to eat in or take out
6. Review the order summary and total price
7. Click **Order Pizza**
8. Confirm the order
9. Use **Reset Form** to create a new order
